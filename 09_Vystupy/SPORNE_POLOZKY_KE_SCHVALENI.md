## 1. ROZPORY MEZI VAULTEM A WEBEM

Následující položky prezentované na webu **tjk.manus.space** nejsou dosud ověřeny proti primárním zdrojům ve vaultu:

### Web dukaz1 – Volební program Bokoče
- **Popis**: Web uvádí volební program Bokoče jako MOTIV pro jednání města. Přesné znění programu není ve vaultu potvrzeno.
- **Problém**: Chybí originální dokument — volební program Bokoče (kandidátka na starostku).
- **Nutná akce**: Získat kopii volebního programu, ověřit tvrzení o vazbě na lyžařský areál.
- **Vault reference**: [[DUK-2026-020_Web_dukaz1_volebni_program_Bokoce]] (pokud existuje)

### Web dukaz4 – Dopis Kučery ignorován
- **Popis**: Web tvrdí, že dopis advokáta Kučery adresovaný městu byl ignorován.
- **Problém**: Vault obsahuje varování Kučery z 26. 1. 2024, ale přesný obsah dopisu, jeho adresát a datum odeslání nejsou ověřeny.
- **Nutná akce**: Získat kopii dopisu, ověřit datum, adresáta a obsah.
- **Vault reference**: [[OSO-2026-015_Mgr_Kucera]]

### Web dukaz6 – Tajná jednání s Lesy ČR
- **Popis**: Web popisuje "tajná jednání" mezi městem/soukromými osobami a Lesy ČR ohledně odkupu dluhu TJK.
- **Problém**: Není známo, jaká jednání proběhla, kdo se jich účastnil, kdy přesně. Zápisy chybí.
- **Nutná akce**: Získat zápisy z jednání, ověřit data a účastníky.
- **Vault reference**: [[ANL-2026-004_Prehled_jednani_Lesy_CR]], [[ORG-2026-008_Lesy_CR]]

### Web dukaz8 – E-mail Tučkové
- **Popis**: Web cituje e-mail od Tučkové z Lesy ČR. Obsah a datum e-mailu nejsou ověřeny.
- **Problém**: Není potvrzeno, kdo je Tučková (přesná role v Lesy ČR), co e-mail obsahuje, kdy byl odeslán.
- **Nutná akce**: Získat kopii e-mailu, ověřit identitu Tučkové.
- **Vault reference**: Tučková (Lesy ČR) — dosud neevidována v [[OSOBY_PREHLED]]

### Web dukaz7 – Nesplněné pověření UR-899
- **Popis**: Web tvrdí, že usnesení rady UR-899-25/25 bylo označeno jako "splněno", přestože žádné skutečné jednání s Lesy ČR neproběhlo.
- **Problém**: Chybí originální dokumentace UR-899 (text usnesení, zápis o splnění).
- **Nutná akce**: Vyžádat UR-899 a související dokumentaci od Města Krupka.
- **Vault reference**: [[ANL-2026-005_Prehled_jednani_Mesto_Krupka]]

### Web – Infografika vazeb
- **Popis**: Web obsahuje vizuální mapu vazeb mezi osobami a subjekty.
- **Problém**: Data v infografice je nutné křížově ověřit proti záznamům ve vaultu — mohou obsahovat nepřesnosti nebo zjednodušení.
- **Nutná akce**: Porovnat každou vazbu v infografice proti [[VYS-2026-002_Infografika_vazeb]] a primárním záznamům.
- **Vault reference**: [[VYS-2026-002_Infografika_vazeb]]
